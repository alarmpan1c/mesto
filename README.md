# Проект: Место
Проект по 4 спринту на курсе по веб-разработке от Яндекс.Практикум. Сайт адаптирован для разных устройств для удобства пользования. 

### Использованы следующие технологии:
* HTML
* CSS
* Flexbox
* Grid Layout
* Адаптация с помощью медиазапросов
* JavaScript
* Методология БЭМ
